Las propiedades min-width y min-height
establecen el tamaño mínimo de un
elemento mediante un valor numérico (con su unidad)
o un porcentaje (con respecto al elemento padre).
width:80%;
min-width:350px

Las propiedades max-width y max-height
establecen el tamaño máximo de un elemento
mediante un valor numérico (con su unidad)
o un porcentaje (con respecto al elemento padre)
width:80%;
max-width:350px;
