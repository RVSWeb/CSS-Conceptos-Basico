El elemento img es un elemento inline-block lo que significa que ocupa el espacio del elemento siguiendo una linea y ademas obecede a los elementos width , height , margin que se le asigne desde el archivo de CSS , es la forma de tener elementos encadenados y ademas definirle su posicion , cuando se le aplica la propiedad float se convierte en un elemento block y se superpone al elemento que tienen por debajo , todos los demas elementos se coloca a su derecha o izquierda
